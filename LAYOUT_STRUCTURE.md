## Animation Timeline

### Page Load Sequence (Hero Section)
```
0.0s: Background gradient appears
0.2s: Floating orbs fade in and start pulsing
0.4s: Name text slides up and fades in
0.6s: Tagline slides up and fades in
0.8s: Buttons slide up and fade in
1.0s: Social icons fade in
1.2s: Scroll indicator starts bouncing
```

### Scroll-Triggered Animations

#### About Section
```
On Enter Viewport (-100px margin):
  - Section title: fade in + slide up (0.8s)
  - Image: slide from left (0.8s, 0.2s delay)
  - Content: slide from right (0.8s, 0.4s delay)
  - Stat cards: scale up (0.3s, 0.6s delay, staggered)
```

#### Skills Section
```
On Enter Viewport:
  - Section title: fade in + slide up (0.8s)
  - Skill cards: fade in + slide up (0.5s, staggered 0.1s)
  - Skill bars: width from 0 to target (1s, 0.5s delay)
```

#### Projects Section
```
On Enter Viewport:
  - Section title: fade in + slide up (0.8s)
  - Project cards: fade in + slide up (0.5s, staggered 0.1s)
  
On Hover (each card):
  - Border color change (0.3s)
  - Shadow elevation increase (0.3s)
  - Slight scale up (1.02x)
  
On Click:
  - Modal: scale from 0.9 to 1.0 (0.3s)
  - Backdrop: fade in (0.2s)
```

#### Contact Section
```
On Enter Viewport:
  - Section title: fade in + slide up (0.8s)
  - Contact info: slide from left (0.8s, 0.2s delay)
  - Form: slide from right (0.8s, 0.4s delay)
  
On Contact Card Hover:
  - Slide right 10px (0.2s)
  
On Form Submit:
  - Button: spinner animation
  - Success: fade in message (0.3s)
```

## Responsive Behavior

### Mobile (< 768px)
- Single column layouts
- Stacked hero buttons
- Simplified animations (reduce motion)
- Larger touch targets (min 44px)
- Hamburger menu if navigation needed

### Tablet (768px - 1024px)
- Two column grid for skills/projects
- Side-by-side about section
- Full animations enabled

### Desktop (> 1024px)
- Three column grid for projects
- Max width container (7xl = 1280px)
- Parallax effects on scroll
- Full animation suite

## Performance Considerations

### Lazy Loading
```jsx
// Components load as they enter viewport
<LazyLoad once offset={100}>
  <ProjectsSection />
</LazyLoad>
```

### Image Optimization
```jsx
// Next.js Image component
<Image
  src="/profile.jpg"
  width={400}
  height={400}
  alt="Profile"
  loading="lazy"
  quality={85}
/>
```

### Animation Performance
```css
/* Use transforms instead of position */
.animate {
  transform: translateY(20px);  /* Good */
  /* top: 20px; */ /* Bad */
}

/* Enable GPU acceleration */
.smooth {
  will-change: transform;
  transform: translateZ(0);
}
```
